🔒 권한 원칙 (기존 메타버스와 동일)

  • DB/RLS 그대로 — 스키마 변경 없음 (migration 34개 재사용)
  • admin ≡ director — 동일 isAdmin 플래그로 완전 동일 취급
  • 차이: 초대 코드 / 차단 로그만 admin 전용 (AdminPanel 기준)
  • leader는 관리 메뉴 노출, 자기 팀만 필터 (UI + RLS 이중 방어)
  • 익명 VOC/쪽지/고민은 admin도 작성자 조회 불가
  • 실명 표시는 admin/director만 (getDisplayName(p, isAdmin))
  • 공지 랜딩은 미확인 긴급 공지 있을 때만 노출
  • v1↔v2 언제든 전환 · 모바일은 기존 MobileLayout 유지
